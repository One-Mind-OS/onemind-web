# How to Switch Between Light and Dark Themes

## Quick Guide for Users

OneMind OS v2 now supports both **Light Mode** (professional, clean design) and **Dark Mode** (tactical, command center aesthetic). You can switch between them at any time!

---

## Switching Themes

### Step-by-Step Instructions

1. **Open the Navigation Drawer**
   - Click the hamburger menu (☰) in the top-left corner
   - Or swipe from the left edge on mobile devices

2. **Scroll to the Bottom**
   - The theme toggle button is in the footer section
   - Above the "OPERATIONAL - ALL SYSTEMS ACTIVE" status indicator

3. **Click the Theme Button**
   - If in Light Mode: Click "DARK MODE" button (with moon icon 🌙)
   - If in Dark Mode: Click "LIGHT MODE" button (with sun icon ☀️)

4. **Theme Changes Instantly**
   - The entire app updates immediately
   - Your choice is saved automatically
   - Next time you open the app, your theme preference is remembered

## Frequently Asked Questions

### Q: Will my theme choice be saved?
**A:** Yes! Your theme preference is automatically saved to your browser's local storage and will persist across sessions.

### Q: Can I set different themes for different devices?
**A:** Yes, each browser/device saves its own theme preference independently.

### Q: What's the default theme?
**A:** Light Mode is the default for first-time users. After that, your last selection is remembered.

### Q: Does the theme affect performance?
**A:** No, theme switching has no noticeable impact on performance. The switch is instant and efficient.

### Q: Can I schedule automatic theme changes?
**A:** Not currently, but this is a planned future enhancement (e.g., auto-dark at night).

### Q: Will my theme sync across browsers?
**A:** Not automatically. Each browser stores its own preference. Cloud sync may be added in future updates.

### Q: Can I customize the colors?
**A:** Not currently, but custom theme support is a planned future enhancement.

### Q: Does the theme change affect saved data?
**A:** No, the theme only affects visual appearance. All your agents, teams, and data remain unchanged.

## Troubleshooting

### Theme Not Saving
**Problem:** Theme resets to default when I close the browser.

**Solutions:**
1. Check if browser is in incognito/private mode (doesn't save preferences)
2. Verify browser allows local storage
3. Clear browser cache and try again
4. Check if you have browser extensions blocking storage

### Button Not Visible
**Problem:** I can't find the theme toggle button.

**Solutions:**
1. Open the navigation drawer (☰ menu)
2. Scroll all the way to the bottom
3. Look for the button above the green "OPERATIONAL" status
4. Try refreshing the page

### Theme Switch Not Working
**Problem:** Clicking the button doesn't change the theme.

**Solutions:**
1. Refresh the page
2. Clear browser cache
3. Check browser console for errors (F12)
4. Try a different browser to isolate the issue

### Colors Look Wrong
**Problem:** Some elements don't match the theme.

**Solutions:**
1. Hard refresh the page (Ctrl+Shift+R or Cmd+Shift+R)
2. Clear browser cache completely
3. Check if browser has high contrast mode enabled
4. Verify you're on the latest version
